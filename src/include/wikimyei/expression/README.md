# Wikimyei Expression

Expression is where Wikimyei gives form to evidence before learned workers use
it.

It is not Ujcamei: it does not fetch or own source input.

It is not representation: it does not learn embeddings.

It is not inference: it does not forecast.

Expression may contain deterministic transformations, graph-aware tensor
formation, and other worker-native ways of shaping evidence.

The first expression is NodeLift.
