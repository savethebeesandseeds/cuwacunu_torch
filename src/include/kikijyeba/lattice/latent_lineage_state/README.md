# Latent Lineage State

Latent lineage state DSL lives under Kikijyeba lattice because it records
runtime memory in a form that can be normalized into lattice state.

Namespace: `cuwacunu::kikijyeba::lattice::latent_lineage_state`.

Runtime report `.lls` helpers live under
`cuwacunu::kikijyeba::lattice::latent_lineage_state::runtime_report`.
